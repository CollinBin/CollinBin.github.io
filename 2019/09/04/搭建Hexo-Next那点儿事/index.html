1、安装 homebrew

  • 可以通过 brew -v 来查看是否安装了 homebrew，如果能正确显示 homebrew 的版本号，说明 homebrew 已安装

  • 如果没有安装 homebrew，通过终端命令安装

2、安装 node

  • 验证 node 是否安装成功
